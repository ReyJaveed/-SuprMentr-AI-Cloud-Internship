# Build a Text Cleaner

## Objective
Write code to remove punctuation, lowercase text, remove stopwords, and test the cleaner.

## Features
- Lowercase all text  
- Remove punctuation  
- Remove common stopwords  
- Works on sample messy sentences

## Concepts Used
- String manipulation  
- Stopwords removal (NLTK)  
- Text preprocessing for NLP

## Output Example
Original: "OMG 😱 I luv this!!!"  
Cleaned: "omg luv"
