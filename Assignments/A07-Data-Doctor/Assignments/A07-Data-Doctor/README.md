# Data Doctor: Dataset Cleaning

## Objective
Clean a dataset by removing duplicates, handling missing values, and standardizing text.

## Features
- Remove duplicate rows
- Fill missing values (with mean)
- Standardize text (capitalize names, cities)

## Concepts Used
- Pandas DataFrame
- drop_duplicates()
- fillna()
- String operations

## Output
Shows cleaned dataset and a summary of cleaning actions.
