# Text Challenges

## Objective
Collect 20 messy sentences, identify slang, emojis, typos, and explain preprocessing needed.

## Features
- Sample 20 sentences with slang, emojis, and typos  
- Preprocessing steps:  
  - Lowercasing  
  - Remove punctuation  
  - Remove emojis / non-ASCII characters  
- Cleaned sentences are output for further NLP tasks

## Concepts Used
- String manipulation
- Regular expressions
- Text preprocessing for NLP

## Output Example
Original: "OMG 😱 I luv this!!!"  
Cleaned: "omg i luv this"
