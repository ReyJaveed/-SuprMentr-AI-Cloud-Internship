# House Price Predictor

## Objective
Train a Linear Regression model to predict house prices based on size.

## Features
- Dataset: House Size vs Price
- Linear Regression model training
- Predict price for new house size
- Display model coefficient and intercept

## Concepts Used
- Pandas DataFrame
- Linear Regression (scikit-learn)
- Feature and label identification
- Predicting output using trained model

## Output Example
Predicted price for a 1600 sq ft house: $370000  
Model coefficient: 200  
Model intercept: 50000
