# Decision Tree on Paper

## Objective
Draw a decision tree to predict whether you should play outside.

## Features / Conditions
- Weather: Sunny, Rainy, Cloudy  
- Temperature: Hot, Mild, Cool  
- Humidity: High, Normal  
- Wind: Strong, Weak  
- Free time available: Yes, No

## Decision Tree Example

1. Check Weather:
   - Sunny → Check Humidity
      - High → Don’t Play
      - Normal → Play
   - Rainy → Don’t Play
   - Cloudy → Check Wind
      - Strong → Don’t Play
      - Weak → Play

## Concepts Used
- Decision Tree
- Conditional branching
- Simple yes/no outcomes

## Output Example
Based on inputs like Weather=Sunny, Humidity=Normal → Decision: Play
