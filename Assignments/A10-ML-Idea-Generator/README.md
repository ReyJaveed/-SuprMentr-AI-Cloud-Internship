# ML Idea Generator

## Objective
Suggest machine learning problems in different domains and describe input → output.

## ML Problem Ideas

1. **College Attendance Prediction**  
   - **Input:** Student class attendance, assignments completed, previous marks  
   - **Output:** Likely attendance percentage

2. **Healthcare: Disease Prediction**  
   - **Input:** Age, weight, symptoms, medical history  
   - **Output:** Probability of specific disease

3. **Shopping: Product Recommendation**  
   - **Input:** Past purchase history, browsing patterns, ratings  
   - **Output:** Recommended products

4. **College: Exam Performance Prediction**  
   - **Input:** Study hours, sleep hours, past marks  
   - **Output:** Predicted exam score

5. **Healthcare: Heart Disease Risk**
