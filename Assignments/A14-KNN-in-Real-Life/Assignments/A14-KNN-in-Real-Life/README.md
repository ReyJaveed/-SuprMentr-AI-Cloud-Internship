# KNN in Real Life

## Objective
Explain Netflix-like recommendations using KNN and demonstrate a small similarity example.

## Features
- KNN finds similar users based on ratings  
- Provides recommendations by finding nearest neighbors  
- Simple Python example included

## Concepts Used
- K-Nearest Neighbors (KNN)
- Cosine similarity
- sklearn NearestNeighbors

## Output Example
Nearest neighbors for User 1:  
Neighbor 1: User 2 at distance 0.12  
Neighbor 2: User 3 at distance 0.87
