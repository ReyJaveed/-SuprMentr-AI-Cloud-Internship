# Storytelling with Graphs

## Objective
Create bar chart, pie chart, histogram, and write a short data story.

## Features
- Bar chart for fruit sales  
- Pie chart showing proportion of sales  
- Histogram for age distribution  
- Short data story explaining trends

## Concepts Used
- Matplotlib library
- Data visualization techniques
- Understanding trends and distribution

## Output Example
- Bar chart shows quantity of each fruit  
- Pie chart shows percentage contribution  
- Histogram shows age frequency  
- Insights summarized in 5 points
