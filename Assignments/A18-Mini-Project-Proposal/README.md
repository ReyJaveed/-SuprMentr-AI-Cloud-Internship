# Mini Project Proposal

## Objective
Submit a 1-page proposal for a mini project including problem, dataset, algorithm, and expected output.

## Project Idea
**Problem:** Predict student performance based on study habits and lifestyle factors.  

**Dataset:** 
- Features: Study hours, sleep hours, past grades, attendance  
- Labels: Predicted exam scores  
- Size: 200–300 students  

**Algorithm:** Linear Regression / Decision Tree  

**Expected Output:** 
- Predict exam scores for new students  
- Insights into which factors affect performance most  
- Visualizations showing relationship between study habits and marks

## Additional Notes
- Dataset will be cleaned and preprocessed  
- Model evaluation metrics: MAE, RMSE  
- Possible extensions: Include more features like participation, extracurricular activities
