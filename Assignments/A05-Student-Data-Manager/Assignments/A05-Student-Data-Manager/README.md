# Student Data Manager

## Objective
Store student data in Python dictionaries and perform analytics.

## Features
- Stores data for 5 students  
- Calculates topper  
- Computes class average  
- Assigns grades based on marks

## Concepts Used
- Python Dictionaries
- Loops
- Functions
- Conditional Statements

## Output Example
Topper: Charlie with 92 marks  
Class Average: 79.20  
Alice → Marks: 85, Grade: A  
...
