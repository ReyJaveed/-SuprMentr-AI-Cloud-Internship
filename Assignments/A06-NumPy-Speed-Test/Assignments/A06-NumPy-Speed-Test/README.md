# NumPy Speed Test

## Objective
Compare execution speed of Python lists vs NumPy arrays for 1 million numbers.

## Features
- Measures time for squaring elements
- Shows speed difference
- Provides key observations

## Concepts Used
- Python Lists
- NumPy Arrays
- Time module
- Vectorized operations

## Observations
1. NumPy operations are much faster.  
2. NumPy arrays use vectorized operations.  
3. NumPy arrays are memory-efficient.
