# Customer Segmentation

## Objective
Perform K-Means clustering on customer data and describe customer groups.

## Features
- Dataset: Customer Age and Annual Income  
- Apply K-Means clustering (3 clusters)  
- Visualize clusters with scatter plot  
- Identify characteristics of each customer group

## Concepts Used
- Pandas DataFrame
- K-Means Clustering (scikit-learn)
- Data Visualization (matplotlib)

## Output Example
CustomerID, Age, AnnualIncome, Cluster  
Clusters show groupings based on age and income
