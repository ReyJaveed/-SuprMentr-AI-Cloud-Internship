# Dataset Detective

## Objective
Load a dataset, explore data, count missing values, and generate insights.

## Features
- View top rows of the dataset
- Identify column with highest value
- Count missing values
- Generate 5 key insights

## Concepts Used
- Pandas DataFrame
- head(), isnull(), sum()
- idxmax(), unique()

## Output Example
Top rows of dataset displayed  
Column with highest value: Marks  
Missing values: 0  
Insights listed for analysis
