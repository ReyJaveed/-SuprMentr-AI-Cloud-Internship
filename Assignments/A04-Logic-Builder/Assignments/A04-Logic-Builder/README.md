# Logic Builder (FizzBuzz)

## Objective
Print numbers from 1 to 50 with Fizz/Buzz logic and count occurrences.

## Rules
- Print "Fizz" for multiples of 3  
- Print "Buzz" for multiples of 5  
- Print "FizzBuzz" for multiples of both

## Features
- Loop through 1–50  
- Count how many times Fizz, Buzz, FizzBuzz appear  
- Demonstrates use of loops and conditional statements

## Output Example
1  
2  
Fizz  
4  
Buzz  
...
